Projects

Floh Store

Floh.Store began with the idea of creating a product that could compete with established services, built using the MERN stack and offering advanced features. Our development process involved capturing ideas in Figma, structuring them into TODO lists, and translating them into code. The backend is secured with JWT, CSRF protection, and cookies, while the frontend uses modern technologies like WebSocket for fast data transmission. The platform is optimized for mobile-first use and works well on ultrawide screens. Available as both a website and an Android app, Floh.Store is the result of intense collaboration and marks the beginning of our journey in software development.

My Calculator

Created a functional calculator website using JavaScript. The calculator supports essential operations such as addition, subtraction, multiplication, and division. Additionally, it includes features for square root and percentage calculations. Robust error handling ensures the calculator responds appropriately to invalid inputs. The calculator utilizes object-oriented programming principles, featuring a Calculator class with methods for performing mathematical operations. It can handle various calculations, including addition, subtraction, multiplication, division, remainder, exponentiation, square root, and percentage calculations. The code also includes checks for dividing by zero and handling negative numbers when calculating square roots. In the interface, buttons trigger corresponding functions to append numbers and operators to the input, enabling users to perform calculations with ease. The calculator's result area dynamically updates as users input expressions, providing real-time feedback. The script also includes console log statements for testing various calculator methods, ensuring their accuracy and reliability. This project highlights my proficiency in JavaScript and my ability to create interactive web applications with error-free mathematical computations.

Slot Game

This project is a simple slot game created to practice JavaScript. Players start with 100 coins and can play by pressing the spacebar. The game features six symbols and randomly selects three each turn. Winning combinations grant coin rewards, while losing deducts coins. The game includes a user interface that displays the current coin count and the results of each spin. The code consists of functions to manage coin addition and deduction, symbol selection, win calculation, and display updates. While not highly complex, this project serves as a foundational exercise in JavaScript programming and DOM manipulation, providing an interactive experience for users.

Offer Section

One of my early attempts at web design involved creating a mobile service offers section that featured responsive cards. These cards were designed to react and move slightly when users hovered over them or selected them, similar to the design of well-known mobile service providers. Each card represented a different mobile service offer, and they were designed to be responsive to different screen sizes and devices. This was achieved using CSS media queries and flexible layout techniques. The cards themselves were designed using a combination of HTML, CSS, and JavaScript. They featured eye-catching graphics, bold text, and clear calls to action. The hover and selection effects were achieved using animations and transitions. Overall, this project was a fun challenge that taught me a lot about responsive web design and how to create engaging and interactive user interfaces. It also helped me to refine my design skills and experiment with new techniques and technologies.

Retro Game

I created a Retro Snake Game using the HTML5 Canvas element. This was one of my first projects, and I pieced together the necessary parts of the code to make it work. The game is a classic iteration of the popular Snake game from the 1970s. The objective is to guide the snake to eat the food on the screen, while avoiding collisions with the snake's own tail. The snake's movement is controlled using the arrow keys on the keyboard. The game increases in difficulty as the snake grows longer and the speed increases. The player can earn points by eating the food, and the game ends when the snake collides with the wall or its own tail. Overall, the game was a fun project to work on, and it allowed me to experiment with the Canvas element and game development in HTML5.
